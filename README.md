### 2️⃣ 🚆 AI-Driven Train Induction & Fleet Optimization System
**Machine learning system optimizing metro train scheduling to save operational costs**

**The Problem:**
Metro systems waste ₹Crores in operational costs due to inefficient fleet scheduling and high idle times.

**The Solution:**
Built ML-based demand forecasting system that recommends optimal train scheduling and reduces wasted fleet capacity.

**Technical Implementation:**
- **Data Pipeline:** Processed **100K+ historical passenger records** using Pandas/NumPy
- **Feature Engineering:** Temporal features (day/hour), seasonal patterns, special events
- **ML Model:** ARIMA + Gradient Boosting
  - **<8% MAPE** (Mean Absolute Percentage Error) on 7-day forecasts
  - Handles peak hours, holidays, special events
- **Backend:** Flask API serving real-time predictions
- **Frontend:** React-based analytics dashboard
  - Visualizes demand patterns
  - Recommends schedule optimizations
  - Shows projected cost savings

**Impact:**
- Projected annual operational savings of **₹2+ Crores** through optimized scheduling
- Deployed recommendation engine used for real scheduling decisions

**Tech Stack:** Python, scikit-learn, ARIMA, Flask, React, MySQL
